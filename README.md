# Site personnel — Mohamed Youssef Litayem

Site personnel simple en HTML, CSS et JavaScript, conçu pour présenter mon profil, mes projets et mes liens principaux.

## Contenu

- Page d'accueil responsive
- Présentation courte
- Projets personnels
- Compétences principales
- Liens vers GitHub et email
- Balises SEO pour le référencement

## Publication avec GitHub Pages

1. Créer un dépôt GitHub nommé `yolita1.github.io`
2. Ajouter les fichiers `index.html`, `style.css` et `script.js`
3. Aller dans **Settings > Pages**
4. Choisir la branche `main`
5. Le site sera disponible sur :

```text
https://yolita1.github.io/
```
